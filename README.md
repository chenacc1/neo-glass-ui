## 核心特性

| 特性 | 说明 |
|------|------|
| **9 层新拟态阴影** | 2 层外层 + 7 层内层，精确定向光源 |
| **遮罩复合描边环** | `mask-composite: exclude` 跟随任意圆角 |
| **3D 透视倾斜** | CSS 自定义属性驱动，弹簧缓动回弹 |
| **玻璃球悬停物理** | `scale(1.15)` + 11 层阴影增强 |
| **水纹焦散纹理** | SVG `feTurbulence` 数据 URI |
| **WebGL 流动波浪** | 全屏着色器背景，鼠标交互，天蓝色调 |
| **零依赖** | 无框架、无构建工具，浏览器直接打开 |
